## 📁 How to Add Images

1. Create the folder: `public/images/`
2. Export/save each image as `.webp` with the exact filename listed above
3. The site will automatically display them — no code changes needed

**Tip:** Use [Squoosh](https://squoosh.app) or [cwebp](https://developers.google.com/speed/webp/docs/cwebp) to convert JPG/PNG to WebP.
